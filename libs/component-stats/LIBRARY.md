# @component-stats/component-stats

A Node.js library that parses Angular projects and analyzes component usage in templates using the Angular compiler and TypeScript.

This library is part of the component-stats Nx monorepo.

## Documentation

See the main documentation files in this directory:
- [README.md](./README.md) - Main documentation
- [USAGE.md](./USAGE.md) - Detailed usage examples
- [QUICKSTART.md](./QUICKSTART.md) - Quick start guide
- [DEVELOPMENT.md](./DEVELOPMENT.md) - Development guide

## Building

From the workspace root:

```bash
nx build component-stats
```

## Testing

```bash
nx test component-stats
```
